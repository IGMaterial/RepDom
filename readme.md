# fhir-do-core

## FHIR Core Dominican Republic Implementation Guide

Este paquete contiene la guía de implementación FHIR Core de República Dominicana, basada en HL7 FHIR R4 y construidas desde cero usando FHIR Shorthand (FSH) y SUSHI.


## Requisitos

- Node.js LTS
- SUSHI
- HL7 IG Publisher
- Java 17+

## Instalación rápida de SUSHI

```bash
npm install -g fsh-sushi
```

## Compilar la Guía Core

```bash
cd fhir-do-core
sushi .
```
